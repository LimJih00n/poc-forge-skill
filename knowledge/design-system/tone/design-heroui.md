# HeroUI Media Reference

이 문서는 OpenAI 생성 UX 화면 이미지의 시각 참고용 메모입니다. Dealcatch 자동 생성 포맷에서는 `04-ux/assets/provided-media/images/`의 생성 이미지를 proposal page에 직접 표시하며, 이미지가 없으면 제공 예정 상태로 둡니다.

## 사용 방식

- HeroUI 계열의 밀도, 버튼/카드/입력/탭 비례, 상태 표현을 이미지 프롬프트의 참고 요소로 사용합니다.
- 실제 proposal page에는 `04-ux/assets/provided-media/images/`의 생성 화면 이미지만 표시합니다.
- 브랜드명, 내부 코드명, 특정 디자인 시스템 이름은 최종 proposal page 문구에 노출하지 않습니다.

## 참고 포인트

- 운영툴/관리자 화면에는 명확한 표, 필터, 상태 배지, 조작 버튼을 우선합니다.
- 소비자 화면에는 첫 viewport의 CTA, 카드 비율, 이미지/텍스트 밀도를 우선합니다.
- 폰트 굵기는 700 이하로 가정하고, 강조는 색상 대비, 크기, 간격으로 조절합니다.
- 동일 프로젝트의 모든 화면 이미지는 하나의 팔레트와 컴포넌트 언어로 이어져야 합니다.
